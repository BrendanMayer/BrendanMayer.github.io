Dungeon Odyssey started as an experiment in building replayable spaces with a branch-based procedural layout. I wanted each run to feel structured enough to be readable while still producing variation.

Core idea

The dungeon is assembled from room and hallway prefabs. Instead of treating the layout as pure randomness, I used branch rules to shape progression and create more intentional paths.

Goals

  • Create readable runs with enough variation to stay interesting
  • Support verticality and room variety
  • Keep generation logic modular for iteration

Takeaway

The most useful lesson was that procedural generation gets much better when the system is constrained in ways that support player understanding.

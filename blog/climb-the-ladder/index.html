Climb The Ladder — Procedural Dialogue + Voice in Unity

Unity C# AI   •   29 Jan 2025

Climb The Ladder is my final‑year project: an office‑set prototype where you talk to co‑workers to get promoted. NPCs respond with procedurally generated dialogue powered by ChatGPT, and you can speak your lines using Whisper for voice recognition.

Why this project?

I’ve been fascinated by AI‑driven NPCs since seeing Tamulur’s 2023 video showcasing ChatGPT characters in a VR tavern. Another inspiration was the indie game Suck Up!, where AI‑generated conversations drive the gameplay loop. I wanted to explore how far that idea could go in a small, focused prototype.

How it works

The project uses a Unity–OpenAI bridge (HTTPS requests) to talk to ChatGPT and Whisper. The main challenge was giving ChatGPT enough context to speak as a specific character in a live game world.

To solve that, I built a personality system with Scriptable Objects—name, job title, interests, dislikes, and more. Each NPC receives a short “loading message” before the first line, seeded with this data plus current game state.

The task system feeds completed tasks into the conversation context too, so characters can refer to what you’ve done. Voice input is recorded, transcribed by Whisper, then sent as text to ChatGPT for a response.

State machines

Player and NPC behaviour runs on finite state machines for clarity and performance. States gate actions—e.g. the player can’t interact while airborne—so scripts only run when relevant.

Challenges

Reliable real‑time integration between transcription and chat required a lot of iteration. I settled on a lightweight model for quick responses and kept interactions one‑to‑one to control API load.

Current status

The prototype supports natural conversations with multiple NPCs, with scalable instances and manageable request volume. At the time of writing, most core systems are in place; I’ll update this post after submission.

Future work

  • A small skill system to unlock social perks and difficulty modifiers.
  • Integrate my 3D procedural office generator for replayable layouts.
